My Recent Projects

Metrysk: GHG Emissions Calculation Platform

Contributed to Metrysk, a real-world platform for calculating and reporting Greenhouse Gas (GHG) emissions. Users can upload financial portfolios, trigger complex GHG calculations, and view detailed summaries and reports. The application was built with a JavaScript stack, featuring a Node.js backend and a React frontend for an interactive user experience.

Vitronic: Component Management System

Developed a robust component management system for Vitronic, a real-world enterprise project. This application leverages advanced architectural patterns including CQRS (Command Query Responsibility Segregation) and Event Sourcing to ensure scalability and data consistency. The backend was built using .NET, with React for a dynamic and responsive frontend. RabbitMQ was integrated for asynchronous messaging and event handling, facilitating decoupled services and improved system resilience.

Portail Acadimic : Application Mobile Scolaire

Développement d'une application mobile **native pour Android**, nommée 'Portail Acadimic', utilisant le langage **Kotlin**. Cette application fonctionne comme un portail d'informations pour les étudiants (ou utilisateurs similaires au sein d'une école). Elle permet notamment la **connexion sécurisée**, la consultation des **devoirs (Assignments)** et **annonces (Announcements)**, l'accès à un module de **chat** pour les échanges, et la visualisation du **profil utilisateur**. Le projet met en pratique le développement mobile natif Kotlin pour offrir une expérience fluide et performante sur Android.

Sakani.com: Comprehensive Real Estate Platform

Sakani.com is a full-featured real estate platform designed to connect buyers, sellers, and renters with their ideal properties. It offers an intuitive user experience, starting with a welcoming landing page and secure authentication. Built with a powerful .NET backend and a dynamic React frontend, Sakani.com demonstrates proficiency in developing complex, end-to-end web applications focused on real-world utility, truly helping users take 'the first step towards their new home.'
