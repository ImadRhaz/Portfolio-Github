My Recent Projects

Metrysk: GHG Emissions Calculation Platform

Contributed to Metrysk, a real-world platform for calculating and reporting Greenhouse Gas (GHG) emissions. Users can upload financial portfolios, trigger complex GHG calculations, and view detailed summaries and reports. The application was built with a JavaScript stack, including a Django backend and a React frontend for an interactive user experience.

Vitronic: Component Management System

Developed a robust component management system for Vitronic, a real-world enterprise project. This application leverages advanced architectural patterns, including CQRS (Command Query Responsibility Segregation) and Event Sourcing, to ensure scalability and data consistency. The backend was built with .NET, with React for a dynamic and responsive frontend. RabbitMQ was integrated for asynchronous messaging and event handling, facilitating decoupled services and improved system resilience.

Acadimic Portal: School Mobile Application

Developed a native Android mobile application, named 'Acadimic Portal', using the Kotlin language. This application functions as an information portal for students. It notably allows for secure login, viewing Assignments and Announcements, accessing a chat module, and viewing the user profile. The project puts native Kotlin mobile development into practice to offer a smooth experience on Android.

Sakani.com: Comprehensive Real Estate Platform

Sakani.com is a comprehensive real estate platform designed to connect buyers, sellers, and renters. It offers an intuitive user experience, with a welcoming homepage and secure authentication. Users can browse a vast catalog of property listings with detailed information. The platform includes robust search and filtering features. Sellers can easily add new listings. Built with a powerful .NET backend and a dynamic React frontend, Sakani.com demonstrates proficiency in developing complex web applications.
